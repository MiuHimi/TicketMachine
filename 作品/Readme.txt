開発環境
Unity
Version 2019.2.7f2

操作説明
・ボタンクリックで金銭投入画面に移行。
・お金をクリックすることで金銭が投入されます。
・現金の場合クリックで一枚、電子マネーの場合はクリックすると一括で支払われます。
・投入した金額が代金以上になったタイミングで投入したお金、お釣り、残ったお金が表示されます。
・「もう一度使用」ボタンを押すとお釣りが所持金リストに足されて再び購入することが可能になります。
・「終了」ボタンを押すとウィンドウが閉じます。

画面イメージ
・左側の画面が券売機、右側の画面が所持金というイメージです。
・普段使用している券売機に似せたかったため、手間がかかりますが一枚一枚投入できるようにしました。
